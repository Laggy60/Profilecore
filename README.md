# ProfileCore

Simple and fast **profile manager** for Minecraft servers.
Built with a focus on performance and clean code.

---

## Features

* Async database system (no lag)
* HikariCP connection pooling
* Automatic player profiles (UUID based)
* Queue system for efficient queries
* Easy to extend

---

## Setup

1. Configure your database in `config.yml`
2. Start your server
3. Done ✅
